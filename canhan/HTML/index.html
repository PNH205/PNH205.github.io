Learn HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Learn the fundamentals and advanced concepts step by step.

1. HTML Basics

Beginner

Learn the fundamental structure of HTML documents, basic tags, and how to create your first web page.

Document Structure Basic Tags Text Formatting Comments

Example:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

2. Links and Navigation

Beginner

Master creating links, navigation menus, and connecting different pages together.

Anchor Tags Internal Links External Links Navigation

Example:

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="https://example.com">External</a></li>
    </ul>
</nav>

Learn CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. Master the art of making beautiful, responsive websites.

1. CSS Fundamentals

Beginner

Learn CSS syntax, selectors, and basic styling properties to transform your HTML.

Selectors Properties Colors Typography

Example:

/* CSS Basics */
h1 {
    color: #333;
    font-size: 2rem;
    text-align: center;
}

.highlight {
    background-color: yellow;
    padding: 10px;
}

2. Layout & Positioning

Intermediate

Master CSS layout techniques including Flexbox, Grid, and positioning.

Flexbox Grid Positioning Float

Example:

/* Flexbox Layout */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

Learn JavaScript

JavaScript brings interactivity to web pages. Learn programming fundamentals and advanced concepts to build dynamic applications.

1. JavaScript Basics

Beginner

Learn variables, data types, functions, and control structures in JavaScript.

Variables Functions Loops Conditions

Example:

// JavaScript Basics
let greeting = "Hello, World!";
console.log(greeting);

function addNumbers(a, b) {
    return a + b;
}

let result = addNumbers(5, 3);

2. DOM Manipulation

Intermediate

Learn to interact with HTML elements, handle events, and create dynamic web pages.

DOM Selection Event Handling Element Creation Style Manipulation

Example:

// DOM Manipulation
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
    document.body.style.backgroundColor = 'lightblue';
});
